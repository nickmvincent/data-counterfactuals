# Site v1 content archive

This directory preserves the Markdown content that powered the earlier
multi-page Data Counterfactuals site.

The files remain verbatim and are intentionally outside Astro's runtime content
directory. `manifest.json` records their SHA-256 checksums, and the repository
test suite verifies that none are lost or changed accidentally.

Future edited or merged essays may move to the external digital garden. Those
publishing decisions do not modify this source archive.
